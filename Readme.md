# Hotel Reservation Backend API

## Project Outline

- users -> book rooms
- admins -> check rsvp, bookings
- Auth & Authorization -> JWT Tokens
- Hotels -> CRUD API
- Rooms -> CRUD API
- Scripts -> Database Management -> Seeding, Migration
